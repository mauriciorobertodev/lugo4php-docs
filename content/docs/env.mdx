---
title: Env
description: Classe que contém as varáveis de ambiente.
icon: PiGear
---

## getGrpcUrl(): string

Retorna a URL do servidor do jogo para o bot se conectar.

```php
$env->getGrpcUrl();
```

## getGrpcInsecure(): bool

Retorna verdadeiro se a conexão com o gRPC for insegura.

```php
$env->getGrpcInsecure();
```

## getBotSide(): Side

Retorna a lado que o time do bot pertence.

```php
$env->getBotSide();
```

## getBotNumber(): int

Retorna o número do bot.

```php
$env->getBotNumber();
```

## getBotToken(): string

Retorna um token. Usado em jogos oficiais.

```php
$env->getBotToken();
```

## Métodos

```php
getGrpcUrl(): string;
getGrpcInsecure(): bool;
getBotSide(): Side;
getBotNumber(): int;
getBotToken(): string;
```
